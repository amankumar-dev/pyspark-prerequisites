# PySpark Prerequisites – Foundation for Distributed Data Engineering

## 📌 Purpose

This repository contains all the prerequisite concepts required before learning PySpark.

The goal is to build strong fundamentals in:

* Python (core + advanced)
* Pandas and DataFrame operations
* SQL and data warehousing
* Distributed computing basics
* Storage formats and big data concepts

This ensures a deep conceptual understanding before working with large-scale distributed systems like Spark.

---

## 🧠 1️⃣ Python Fundamentals

Covered Topics:

* Python basics and dynamic typing
* Data structures (list, tuple, set, dictionary)
* Mutability and memory concepts
* Functions, `*args`, `**kwargs`
* Lambda functions
* Decorators and generators
* `yield` vs `return`
* OOP concepts (inheritance, polymorphism, encapsulation)
* Method overloading vs overriding
* Exception handling and custom exceptions
* List & dictionary comprehensions
* `map()`, `filter()`, `reduce()`
* Multithreading vs multiprocessing
* GIL (Global Interpreter Lock)
* Serialization

---

## 📊 2️⃣ Pandas & Data Handling

Covered Topics:

* Series vs DataFrame
* GroupBy and aggregation
* Handling missing values
* Merge vs concat
* Indexing
* Vectorization
* Performance limitations of Pandas
* Why Pandas fails with large datasets

* ETL concepts
* Data warehouse fundamentals
